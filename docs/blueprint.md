# **App Name**: Pondara Portfolio

## Core Features:

- Project Showcase: Showcase selected projects with a blueprint-inspired design, linking to 'portfolio' and 'react-projects' repositories.
- Skills Display: Display Pondara's skills using a visually appealing layout with blueprint/neon elements.
- Contact Form: A contact form for visitors to send messages directly to Pondara.

## Style Guidelines:

- Primary colors: Shades of cyan, white, and deep blue to create a blueprint effect.
- Accent color: Neon orange (#FF4500) or pink (#FF69B4) for glowing contrasts.
- Bold, modern, tech-inspired fonts to enhance the futuristic feel.
- Isometric illustrations of tech elements with a blueprint sketch style that glows subtly.
- Clean and minimalistic layout with a blueprint grid background.
- Subtle animations like glowing hover effects on buttons and project cards.

## Original User Request:
Design a modern portfolio website for a web developer named Pondara, inspired by a glowing blueprint paper style. The website should feature a futuristic, neon-glowing aesthetic with a blueprint grid background, using shades of cyan, white, and deep blue for the blueprint effect, and vibrant accents like neon orange or pink for a glowing contrast. Include isometric illustrations of tech elements (like a laptop, code snippets, or a stylized project showcase) with a blueprint sketch style that glows subtly. The layout should be clean and minimalistic, with sections for an introduction ('Hi, I’m Pondara, a Web Developer'), a projects showcase (highlighting repositories like 'portfolio' and 'react-projects'), skills (e.g., React, JavaScript, HTML/CSS), and a contact form. Use bold, modern typography with a tech-inspired font, and add subtle animations like glowing hover effects on buttons and project cards. Ensure the design feels professional, creative, and visually striking, with a balance of blueprint technicality and modern vibrancy."
